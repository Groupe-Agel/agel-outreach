# Changelog

This folder records monthly summaries of shipped work in `agel-outreach`.
One file per month, named `YYYY-MM.md`. Inside each file, weeks are
ordered most recent first, and each week is broken into the same four
sections: **Features**, **Fixes**, **Refactors**, **Chores**.

This index lists the existing month files. Add a new row when you start
a new month.

## How to add a month

1. Create `YYYY-MM.md` in this folder.
2. Open with the standard front matter (see any doc under `../../`).
3. Inside, list weeks in reverse-chronological order with the four
   subsections above.
4. Add a row to the index table above in the same PR.
